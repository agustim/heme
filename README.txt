README
------

HElp ME to install a node

Un sistema per ajudar a l'instal·lador a posar la antena.

INSTALL

copy all file in /tmp and execute install.sh

USE

1. Prepara la antena apuntant al nord. 
2. Anar a la web de node http://<node>/heme
3. Fer clic al boto "Spectrum analysis".
4. Anar girant la antena a raó de 22.5 (La meitat de 45 Graus). Quan els leds ho indiquin.
5. Quan ho hem fet 16 vegades. Ja podem veure el resultat.
